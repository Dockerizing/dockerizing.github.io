Available DLD Images

We are providing already some docker images for the usage in the dockerized linked data infrastructure. They are available github and at the docker registry-hub (watch out for the dld- prefix)

  • Virtuoso 7 Container aksw/dld-store-virtuoso7 docker registry-hub, @github
  • OntoWiki Container aksw/dld-present-ontowiki docker registry-hub, @github
  • Pubby Container aksw/dld-present-pubby docker registry-hub, @github
  • Fuseki Container aksw/dld-store-fuseki docker registry-hub, @github
  • Virtuoso Import Container aksw/dld-load-virtuoso docker registry-hub, @github
  • Virtuoso Backup Container aksw/dld-backup-virtuoso docker registry-hub, @github
  • SNORQL Container aksw/dld-present-snorql docker registry-hub, @github
